Welcome to the NgRx Online Store!

Overview

This is a workshop designed to learn the basics of the Redux pattern and NgRx.

Below are the technologies used in this web app:

  • Angular (v14)
  • Angular Material (v14)
  • RxJS (v7)
  • NgRx (v14)

There are two main pages in this workshop:

  1. Products: A list of products that a user can add to their cart.
  2. Cart: Products the user has selected for purchase.
